1. État : Veille (Idle / Attente)Quand l'IA est allumée mais que personne ne lui parle. Le but est de montrer qu'elle est "en vie" sans être distraite.Corps : Respiration subtile (le torse se gonfle et se dégonfle légèrement). Le poids du corps bascule très lentement d'une jambe à l'autre toutes les 10 à 15 secondes pour éviter un effet "statue".Visage : Clignement des yeux aléatoire (Blink blendshape). Le regard balaie lentement l'environnement ou fixe la caméra de manière détendue.Bras : Le long du corps, ou mains croisées nonchalamment devant elle.2. État : Écoute ActiveQuand le micro détecte ta voix et qu'elle enregistre ton prompt.Corps : Se penche très légèrement en avant vers la caméra.Visage : Verrouille le contact visuel (Eye Tracking activé à 100% sur toi). Les sourcils se lèvent légèrement.Tête : De petits hochements de tête subtils et aléatoires pour signifier "Je t'écoute, continue". Tête légèrement inclinée sur le côté pour un côté plus empathique.3. État : Réflexion (Processing)Quand le prompt est envoyé et que l'IA génère sa réponse. Il faut montrer qu'elle "calcule".Corps : Bras croisés, ou une main qui remonte vers le menton (pose classique de réflexion).Visage : Le regard se détache de la caméra et pointe vers le haut (à droite ou à gauche). Les yeux font de petites saccades rapides. Les sourcils se froncent très légèrement (Blendshape Angry ou Sorrow mixé à 20% pour marquer la concentration).Bouche : Légèrement pincée ou asymétrique.4. État : Recherche de Données (Action spécifique)Si tu intègres une fonction où elle cherche sur le web ou dans une base de données.Corps : Elle lève une main et tapote une oreillette imaginaire (façon agent secret), ou elle lève la main devant elle comme si elle manipulait un écran holographique invisible.Visage : Regard fixe "dans le vide".Détail technique : Si ton shader le permet, c'est le moment de faire briller ses yeux ou le logo "ASTRAL" de sa tenue pour montrer le flux de données !5. État : Parole (Speaking)Quand l'IA te lit la réponse.Visage : Synchronisation labiale activée (Lip-sync basé sur l'audio avec les Blendshapes A, I, U, E, O). Les sourcils bougent en fonction de l'intonation de la voix. De légers sourires réguliers (Blendshape Joy).Corps : Gestuelle des mains (Beat gestures). Les mains s'ouvrent pour expliquer, comptent sur les doigts pour les listes, et accompagnent le rythme de la voix.6. État : Réactions (Triggers bonus)Petites animations courtes pour rendre l'interaction plus humaine.Succès / Salutation : Petit signe de la main joyeux, grand sourire, clin d'œil.Erreur / Incompréhension : Haussement d'épaules, tête penchée sur le côté, regard désolé, main derrière la nuque (style anime).
