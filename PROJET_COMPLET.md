## 📝 NOTES IMPORTANTES

### Sons manquants :

Les fichiers `notification.mp3` et `alarm.mp3` sont des placeholders.

**Pour ajouter les vrais sons :**

1. Trouve des sons libres de droits
2. Place-les dans `assets/sounds/`
3. Noms exacts : `notification.mp3` et `alarm.mp3`

### Firebase Config :

**⚠️ CHANGE LES CREDENTIALS FIREBASE !**

Dans `js/core/firebase-config.js`, remplace par tes propres credentials.

### Dev Keys :

Ajoute les clés dans Firestore (voir `docs/DEV_KEYS_GUIDE.md`).
